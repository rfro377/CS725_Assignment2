7. TEST REQUIREMENTS

No specific test requirements for compliance with this profile are currently defined. However, a set of test panels has been developed that may be used for testing the interoperability provisions of this Compliance Profile. The configurations described in the test panel documentation are executable with the Function Block Development Kit (FBDK), and are located in the /src/ita/ subdirectory of the FBDK download.

The subdirectories of the FBDK src directory contain a large number of library elements compliant to the provisions of IEC 61499-2, which can be used for testing the ability of software tools to comply with the XML import portion of the portability provisions of this Compliance Profile. Similarly, the FBEditor application of the FBDK can also be used to import library elements produced by other software tools to test their compliance with the XML export portability requirements.

Many of the examples provided with the FBDK can also be used for testing the compliance of remote devices with the configurability provisions of this Compliance Profile. Finally, system configurations are provided in the src/ita/ subdirectory of the FBDK for manual testing of configurability.
