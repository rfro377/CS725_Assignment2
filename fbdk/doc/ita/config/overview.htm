IEC 61499 Configurability Test Configurations

The following system configurations, which can be exercised from the Function Block Development Kit (FBDK), provide facilities for manually testing the configurability provisions of the IEC 61499 Compliance Profile for Feasibility Demonstrations. The configurations described in this documentation are executable with the FBDK, and are located in the src/ita/ subdirectory of the FBDK download.

  • MGR_TEST - This system configuration provides facilities to test the functionality of the management commands and responses required by the Compliance Profile.
  • MGR_TEST2 - This system configuration provides facilities to test the functionality of management command and responses in a device with multiple resources.
