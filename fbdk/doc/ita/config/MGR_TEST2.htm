SYSTEM MGR_TEST2

This system configuration provides facilities to test the functionality of the management commands and responses required by the Compliance Profile, in a remote device with multiple resources. The table below shows a typical testing session using this configuration.

Open the DUT (Device Under Test) using the Run option of its pop-up menu. In this case the remote device (an instance of RMT_DEV) will not be visible; its test panel will be a separate instance of FRAME_DEVICE.
Launch the test configuration using the Run button on the FBEditor's menu bar (you may need to use desktop controls to make the DUT visible after starting).
Verify that the DUT correctly implements the addition and counter functions.
Use the QUERY command to verify that DUT.RES1 contains a FB named AD.
Verify that DUT.RES2 does not contain a FB named AD.
Delete DUT.RES.AD and verify the deletion (to save space, only the textual command and response are shown here). 
RES1:<Request ID="3" Action="DELETE" >
  <FB Name="AD" Type="FB_ADD_INT" />
</Request>
<Response ID="3" />
RES1:<Request ID="4" Action="QUERY" >
  <FB Name="AD" Type="*" />
</Request>
<Response ID="4" Reason="NO_SUCH_OBJECT" />
Create a new FB of type FB_SUB_INT and its associated connections in DUT.RES1. 
RES1:<Request ID="6" Action="CREATE" >
  <FB Name="SBTR" Type="FB_SUB_INT" />
</Request>
<Response ID="6" />
RES1:<Request ID="7" Action="CREATE" >
  <Connection Source="SB.IND" Destination="SBTR.REQ" />
</Request>
<Response ID="7" />
RES1:<Request ID="8" Action="CREATE" >
  <Connection Source="SB.RD_1" Destination="SBTR.IN1" />
</Request>
<Response ID="8" />
RES1:<Request ID="9" Action="CREATE" >
  <Connection Source="SB.RD_2" Destination="SBTR.IN2" />
</Request>
<Response ID="9" />
RES1:<Request ID="10" Action="CREATE" >
  <Connection Source="SBTR.CNF" Destination="PUB.REQ" />
</Request>
<Response ID="10" />
RES1:<Request ID="11" Action="CREATE" >
  <Connection Source="SBTR.OUT" Destination="PUB.SD_1" />
</Request>
<Response ID="11" />
Verify that DUT.RES1 now performs integer subtraction (note that it is not possible to change the "+" label in the DEV_TESTER device, since it is not a remote device and does not contain a management resource).
