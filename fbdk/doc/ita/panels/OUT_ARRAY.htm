FUNCTION_BLOCK OUT_ARRAY

An instance of this function block type encapsulates a single-line text field for display of an ARRAY value. The data type of the array elements is specified by the TYPE input, and the number of elements is specified by the N input.

  • See the documentation of the TEST_ARRAY_IO system configuration and the TEST_ARRAY application for examples of the application of instances of this type.
