SYSTEM TEST_ARRAY_IO

This system configuration provides a means for testing the operation of coupled instances of the IN_ARRAY and OUT_ARRAY function block types. Operation of this test configuration is illustrated in the table below. For further details of the operation of these blocks, see the TEST_ARRAY application.

The initial values before transferring any data.
The results after pressing the Enter key in the top window.
Note that the array is filled with the default initial value.
The results after Entering a full array of data.
The results after Entering an extra array element.
Note the truncation to N elements.
The results after Entering fewer than N elements.
Note that the remaining elements are unaffected.
