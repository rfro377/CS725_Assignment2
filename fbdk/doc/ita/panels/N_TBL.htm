FUNCTION_BLOCK N_TBL

An instance of this function block type selects the N-th element from a list of n input strings. If the inequality -1<N<n is not satisfied, the output is an empty string.
  • See the CHOICE_TBL documentation for an example of the use of an instance of this type.
