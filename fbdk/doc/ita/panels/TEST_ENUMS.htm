APPLICATION TEST_ENUMS

This application is part of the CODEC_TEST2 system configuration. It utilizes instances of IN_ENUM and OUT_ANY for input and output of transmitted data, respectively. This data is of the enumerated data types ANALOG_SIGNAL_TYPE and ANALOG_SIGNAL_RANGE defined in Tables 12 and 14 of IEC 61131-3, respectively and documented below.

In the example shown below, three pairs of values of the types ANALOG_SIGNAL_TYPE and ANALOG_SIGNAL_RANGE have been transmitted with ASN.1 tags of CC and CB hexadecimal, respectively.
