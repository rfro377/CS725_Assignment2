APPLICATION TEST_UINTS

This application is part of the CODEC_TEST system configuration. In this example, values of USINT#1, UINT#2, and UDINT#4 have been transmitted with ASN.1 tags of 46, 47 and 48 hexadecimal, respectively. See TEST_BITS for further documentation of the operation of these test panels.
