APPLICATION TEST_ARRAY

This application is part of the CODEC_TEST system configuration. It utilizes instances of IN_ARRAY and OUT_ARRAY for input and output of transmitted data, respectively.

See TEST_BITS and MONITOR for further documentation of the operation of these test panels.

In this example, an array (ASN.1 tag of 76 hexadecimal) of 5 elements of UINT type (ASN.1 tag of 47 hexadecimal) is transmitted.

The initial situation before data transmission.
Data transmission is triggered by pressing the Enter key in the text field or clicking the ">>" button.
Note that the array is filled with the default initial value (0).
The results after transmitting a full array of data.
The results after Entering an extra array element.
Note that only N elements are passed from the IN_ARRAY block for transmission.
The results after Entering fewer than N elements.
Note that the remaining elements are transmitted unchanged.
