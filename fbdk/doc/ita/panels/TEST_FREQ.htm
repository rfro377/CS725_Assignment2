APPLICATION TEST_FREQ

This application is part of the CODEC_TEST2 system configuration. It utilizes instances of IN_ANY and OUT_ANY for input and output of transmitted data, respectively. This data is of the directly derived data type FREQ defined in Table 14 of IEC 61131-3 and documented below. In the example shown below, two values of this data (5.0 and 1.0 respectively) have been transmitted with an ASN.1 tag of CD hexadecimal.
