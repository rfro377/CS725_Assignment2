SYSTEM CS21_TEST

This system configuration tests instances of the CLIENT_2_1 and SERVER_1_2 types, which may be used in the implementation of remote management services as described in subclause 4.3 of the Compliance Profile for IEC 61499 Feasibility Demonstrations. This configuration may be modified to test other client/server pairs or data types.

This is the situation with initial values shown in the IN_ANY and OUT_ANY text fields.
This is the situation after transmitting data by pressing the Enter key in the first field of the CLT window.
This is the situation after entering new data in all the input fields and pressing the Enter key in the second field of the CLT window. Note that the data in the first field of the SRVR window is not updated because the data in the first field of the CLT window was not updated by an Enter keypress.
This is the situation after entering new data in the bottom field of the SRVR window, then pressing the Enter key in the first field of the CLT window, causing an update of all data.
