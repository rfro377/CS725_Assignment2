FUNCTION_BLOCK CHOICE_TBL

This user interface element utilizes an instance of the N_TBL type to select an output from an array of string values. The output is selected from the array position corresponding to the position of a string selected from an input array of string values by an instance of the IN_CHOICE type.

See the MONITOR device configuration for an example of the use of an instance of this type.
