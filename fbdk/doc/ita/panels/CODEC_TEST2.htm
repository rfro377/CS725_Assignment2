SYSTEM CODEC_TEST2

The system configuration shown below tests the encoding and decoding of the IEC 61131-3 derived data types required by the Compliance Profile for IEC 61499 Feasibility Demonstrations . This is done through pairs of panels designed to test the various families of elementary data types, supplemented by a MONITOR panel containing instances of the CHOICE_TBL, UDP_LISTEN and OUT_TEXT types to enable monitoring of the corresponding UDP multicast channels to check for correct encoding.

Each pair of panels is represented both by its device configurations and its corresponding application. The connections involved in distributable event and data flows are shown for each application. Initialization paths, parameter connections, and auxiliary graphical elements such as labels may or may not be shown.

This is an interactive map. Each panel is linked to documentation of its respective application.
