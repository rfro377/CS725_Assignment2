IEC 61499 Interoperability Test Panels

The following system configurations, which can be exercised from the Function Block Development Kit (FBDK), provide test panels for testing the interoperability provisions of the IEC 61499 Compliance Profile for Feasibility Demonstrations. The configurations described in this documentation are executable with the Function Block Development Kit (FBDK), and are located in the src/ita/ subdirectory of the FBDK download.

  • CODEC_TEST - This set of panels tests the proper operation of PUBLISH and SUBSCRIBE blocks over the full range of multicast UDP addresses, and tests the encoding and decoding of all the IEC 61131-3 elementary data types required by the Compliance Profile.
  • CODEC_TEST2 - This set of panels tests the encoding and decoding of all the IEC 61131-3 derived data types with PUBLISH and SUBSCRIBE blocks.
  • CS21_TEST - This set of panels tests the proper operation of CLIENT_2_1 and SERVER_1_2 blocks.
