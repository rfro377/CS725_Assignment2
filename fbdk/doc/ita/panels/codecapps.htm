APPLICATION TEST_INTS

This application is part of the CODEC_TEST system configuration. In this example, values of SINT#1, INT#2, DINT#3 and LINT#4 have been transmitted with ASN.1 tags of 42, 43, 44 and 45 hexadecimal, respectively. See TEST_BITS for further documentation of the operation of these test panels.
