DEVICE MONITOR

This device configuration (shown below) contains an instance of the OUT_TEXT function block type to display the contents of messages received by an instance of the UDP_LISTEN type, as illustrated above. The UDP channel to be monitored is selected by an instance of the CHOICE_TBL type. Additional logic and user interface elements are provided by instances of the OUT_ANY, E_SR and RADIO_BOOL types.

See the documentation of the TEST_BITS application for examples of the use of this device configuration.
