APPLICATION TEST_TIMES

This application is part of the CODEC_TEST system configuration. In this example, values of t#0.001ms and DT#1969-12-31-19:00:00.001 have been transmitted with ASN.1 tags of 4C and 4F hexadecimal, respectively. Note that values of the TIME type are transmitted as a 64-bit signed integer number of microseconds, and values of the DATE_AND_TIME type are transmitted as a 64-bit signed integer number of milliseconds since 1970-01-01-00:00 UTC (Universal Coordinated Time). In this example, the UTC is 1970-01-01-00:00.001 and the local time is UTC-5.

See TEST_BITS for further documentation of the operation of these test panels.
