7. Perform Distribution Design

This system configuration illustrates a distribution design of the TESTBED_MVCL example in which the Model+View functions are located in a remote device named MODEL and the Control+HMI functions are in a local device named CONTROL. Each of the MODEL and CONTROL devices contains a SENSORS resource and an ACTUATORS resource with the following functionality:

Resource
 Function
MODEL.SENSORS
 Gather sensor data and publish it as a WORD.
CONTROL.SENSORS
 Subscribe to sensor data and publish it locally to Control and HMI resources.
MODEL.ACTUATORS
 Subscribe to actuator data and publish it locally.
CONTROL.ACTUATORS
 Gather actuator data from Control and HMI resources and publish it as a WORD.

Bit positions of the packed sensor and actuator data are documented here.
