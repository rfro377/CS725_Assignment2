FUNCTION_BLOCK SORT_CTL

This function block type provides the logic for sortation of a metallic peg from a non-metallic ring, with "anti-jamming" interlocking to prevent the deflection of a ring into a chute where a ring is already present. A RING_LOST output is also provided for counting the number of cases in which the "anti-jamming" feature must be applied. See the TESTBED_MVCL system configuration for an example of its use.
