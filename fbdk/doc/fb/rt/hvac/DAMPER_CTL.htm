FUNCTION_BLOCK DAMPER_CTL

An instance of this function block type implements OPEN/CLOSE control of a bidirectional motor-driven HVAC damper valve. Control action is based on the linear regression parameters of the trend line of the noisy sensor data, derived from an instance of the REGRESS function block type. Deadband based on the estimated standard deviation DEV of the data is used to avoid excessive wear of the motor gearing and valve stem.

See the Java source code for details of the operation of this function block type.

See the VAV_MVCL system configuration for an example of the use of an instance of this type.
