FUNCTION_BLOCK VAV_POST

An instance of this function block type is used in the composite VAV_MDL type to convert a damper position POS into a volumetric FLOW value and a corresponding simulated A/D converter value, proportional to the square of the FLOW value. Random noise of a magnitude determined by the FV parameter is injected into both output values to simulate variations in flow due to pressure perturbations in the supply air duct and the zone being heated or cooled.

Implementation details of this function block type are shown in its Java source code.
