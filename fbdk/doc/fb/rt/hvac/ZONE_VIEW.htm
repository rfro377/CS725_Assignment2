FUNCTION_BLOCK ZONE_VIEW

An instance of this function block type provides a live textual display of the input values TEMP, FLOW, STEMP, XTEMP and LOAD as documented below. The order in which these values are displayed is FLOW, STEMP, TEMP, XTEMP and LOAD, respectively, as illustrated at right.
