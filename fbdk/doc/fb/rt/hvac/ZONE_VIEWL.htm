FUNCTION_BLOCK ZONE_VIEWL

  • An instance of this function block type encapsulates an instance of the ZONE_VIEW type and the local multicast service interfaces necessary for inter-layer communication with the View layer of the MVC framework.
  • See the VAV_VIEWL configuration for examples of the use of instances of this type.
  • For convenience, the names of the local multicast channels used by an instance of this type with an instance name denoted by <name> will have the default values shown in the following table. These values can be overridden by connection to a non-default variable or configuration parameter.

Channel ID Default value Usage
REND_ID "<name>_REND" FLOW input to the ZONE_VIEW block
PARAMS_ID "PARAMS" FLOW, STEMP, XTEMP, LOAD inputs to the ZONE_VIEW block
