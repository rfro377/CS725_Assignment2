FUNCTION_BLOCK ZONE_MDLL

An instance of this function block type adapts an instance of the ZONE_MDL type for use with the local multicast design pattern. The default value of the PARAMS_ID input is "PARAMS"; this is the local multicast channel ID to receive the TI, TA and H parameters of the ZONE_MDL block.
