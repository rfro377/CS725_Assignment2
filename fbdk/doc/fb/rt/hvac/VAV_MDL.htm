FUNCTION_BLOCK VAV_MDL

An instance of this function block type utilizes instances of the VAV_PRE and VAV_POST types to adapt an instance of the MECH2_MDL type for simulation of the operation of a damper valve and the resulting air flow through a Variable Air Volume (VAV) unit.
