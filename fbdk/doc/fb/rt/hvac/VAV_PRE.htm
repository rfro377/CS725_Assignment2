FUNCTION_BLOCK VAV_PRE

An instance of this function block type performs the pre-calculations necessary to combine damper actuation time TACT and simulation clock period DT to produce a motion velocity figure V as input to an instance of the MECH2_MDL type in an instance of the composite VAV_MDL type.
