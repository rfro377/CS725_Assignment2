Preferences Dialog

This dialog is opened with the Preferences button of the FBEditor's menu bar.

The first time this dialog is opened, it shows the values loaded from the editor.ini file (if no value is given in editor.ini for a parameter, the default value is used). The values shown below are those provided by the editor.ini file supplied with the FBDK, which work with the Windows™ XP system on which the FBDK was tested.
