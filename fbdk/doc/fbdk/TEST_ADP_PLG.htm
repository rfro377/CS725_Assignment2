FUNCTION_BLOCK TEST_ADP_PLG

This composite function block type is a provider for an adapter connection of type TEST_ADP, that is, it contains a plug of type TEST_ADP. It contains GUI (graphic user interface) elements that enable it to function as a server for the testing of the adapter connection. See the TEST_ADP_SYS configuration for an example of its use.
