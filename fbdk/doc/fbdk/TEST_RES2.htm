RESOURCE_TYPE TEST_RES2

This resource type is used to demonstrate the way in which test configurations can be used for the testing of resource and device types. A sample test configuration is given as TEST_SYS2.
  • An instance of this resource uses the local multicast design pattern to provide the two inputs to and one output from an instance of the FB_ADD_REAL function block type.
