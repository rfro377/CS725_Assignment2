SYSTEM TEST_ADP_SYS

The system configuration shown below utilizes instances of the TEST_ADP_PLG and TEST_ADP_SKT types to provide graphic user interfaces for testing plug and socket instances, respectively, of the TEST_ADP adapter type. A typical test sequence is illustrated below.

1. The initial situation. 2. The situation after entering data in the TEST_ADP_SKT instance and pressing the REQ pushbutton. 3. The situation after entering data in the TEST_ADP_PLG instance and pressing the CNF pushbutton. 4. The situation after pressing the Restart/Cold menu item.
