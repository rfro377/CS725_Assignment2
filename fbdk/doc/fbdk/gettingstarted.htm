Getting Started

  1. This software enables you to build and test data types, function block types, resource types, device types and system configurations according to the IEC 61499 standard.
  2. For additional information on the use of this software, see the online documentation (also included with the download and accessible from the software's Help menu).
  3. Read the FBDK Terms and Conditions.
  4. If you agree with the Terms and Conditions, download the FBDK.
  5. (Optional) If you have installed a previous version of the FBDK, rename the directory in which it is located, e.g., from C:\fbdk to C:\fbdk_old.
  6. Extract the FBDK from the fbdk.zip file that you downloaded in Step 4 to a home directory, e.g., C:\fbdk\.
  7. (Optional) If you performed the optional Step 5, copy your personal directories and files as necessary from the old home directory to the new home directory.
  8. Download and install JDK 6.
  9. Configure the editor.ini and fbeditor.bat files as necessary.
  10. Launch the FBEditor by following this procedure.
  11. This software has been tested on Windows™ XP Service Pack 3 and Ubuntu® Linux®.
