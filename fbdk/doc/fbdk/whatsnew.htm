What's New

This page documents changes since the 20080101 release of the FBDK.

FBDK [FBRT] [MVCD]

Features

  • The FBDK has now been tested for compatibility with Windows™ XP and Ubuntu® Linux®.
  • The most frequently used operations have been moved to buttons on the menu bar for quick access.
  • The Save As... dialog now includes a PNG option to save the contents of the graphics Worksheet in Portable Network Graphics (PNG) format.
  • A new Preferences dialog has been added to simplify the configuration process.
  • Configurable Look and Feel has been added, including the new Nimbus LAF in the Java™ SE 6 update 10 RC release.
  • Multi-element selection and dragging is now supported.
  • Self-cycles and guard-only cycles in an ECC are now detected in the Transition Dialog.
  • The current directory location now persists between successive invocations of the file Open dialog.
  • Additional Java™ documentation is now generated when using the File Save As Java option.
  • You no longer have to exit and restart the FBEditor in order to test a new FB type after saving it to a new package (you still have to restart after changing and re-saving an existing type).
  • Help information has been updated.

Bug fixes

  • Self-cycles on an EC state are no longer accidentally hidden behind the state upon creation.
  • Adding an event input or output to an adapter type no longer causes a ClassCastException.

FBRT [FBDK] [MVCD]

Features

  • A new PUT_WORD block has been added to the math package for setting or resetting individual bits in a 16-bit WORD.
  • M_INV (matrix inversion) has been replaced by the more general-purpose M_SOLVE (matrix equation solver) in the math/matrix package.

Bug fixes

  • None.

Other Changes

  • FB_INT_TO_BCD_WORD has been simplified.
  • Documentation of launcher.ini has been updated.
  • CLIENTW and SERVERW have been deleted since they are not used in any applications.

MVCD [FBDK] [FBRT]

Features

  • No new features.

Bug fixes

  • None.
