The procedures described below have only been tested with the Windows™ XP operating system and may have to be modified for other operating systems.

Starting the Launcher

  • To launch a system configuration in a *.sys file, or launch a tester window on an instance of an FB type in a *.fbt file, you can drag the icon of the file onto the icon of the launcher.bat file in the FBDK home directory.
    • This is easier if you create a shortcut to launcher.bat on your desktop (you can use the icon in the FBDK_Starter.exe file).
    • In order to test an FB type, it must be already compiled into a *.class file, either in the existing fbrt.jar or in the FBDK's lib directory by using the File Save As...Java option in the FBEditor application.
  • The launcher.bat file can also be invoked in a command prompt window, for instance:
    cd\fbdk
    launcher .\src\student\X2Y2.fbt

The launcher.ini File

Each line of this file consists of a parameter name followed by a parameter value. The allowed parameter names and values are given in the following table. If no value for a parameter is given in the launcher.ini file the default value applies. Different values for these parameters may need to be given for different Java™ Development Kit (JDK™) and Java Runtime Environment (JRE) installations and for non-Windows operating systems.

Parameter Name Explanation Default
bounds The bounding rectangle x,y,width,height of the visible frame (if any) of the launched configuration. 100,100,200,200
timeout The initial timeout period for management commands to remote devices. t#2s
