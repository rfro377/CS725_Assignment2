FUNCTION_BLOCK REGRESS

An instance of this function block type performs linear regression on a series of N sampled values of the input variable IN. The sampling is assumed to occur at uniform time intervals, triggered by the REQ event input. Upon the occurrence of an INIT event, the sample buffer is emptied and the QO output remains false until the first N samples have been taken, and thereafter remains true until the next occurrence of an INIT event.

See the Java source code for details of the operation of this function block type.
