FUNCTION_BLOCK OUT_BOOL

An instance of this function block type utilizes an instance of the FB_SEL_COLOR type and an instance of the OUT_COLOR type to display a colored circle with a label to its right.
  • Upon an occurrence of an event at the REQ input, the color is updated to the value specified by the CO or C1 input depending on whether the value of the IN input is FALSE or TRUE, respectively.
  • See OUT_EVENT for an example of the use of an instance of this function block type.
