FUNCTION_BLOCK E_SR

This function block type represents a simplification of the E_SR event-driven bistable shown in Annex A of IEC 61499-1. It will work in any implementation that does not latch input event states and that ignores any events when the ECC is not in a receptive state. In this case, this behavior equates to ignoring R in the Q0 state and ignoring S in the Q1 state.
