RESOURCE_TYPE PANEL_RESOURCE

An instance of this resource type provides an instance of a Java™ JPanel that can be contained in a JFrame e.g. as provided by an instance of the FRAME_DEVICE type. The associated panel can then serve as a container for JComponent instances that are encapsulated in appropriate instances of function block types such as FB_LABEL.
