FUNCTION_BLOCK OUT_QO_US

An instance of this function block type provides elements for the display of the CNF event, QO and STATUS outputs of a simple function block as described in Annex D.2 of IEC 61499-1.

STATUS values correspond to the constants defined in the FBStatus interface.
