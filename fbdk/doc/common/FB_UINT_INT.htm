FUNCTION_BLOCK FB_UINT_INT

[Viewer Help]
An instance of this function block type converts a variable of IEC 61131-3 type UINT to a variable of IEC 61131-3 type INT. The output range is limited to the range for variables of type INT defined in IEC 61131-3.
