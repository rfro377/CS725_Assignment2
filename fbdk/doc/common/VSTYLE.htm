TYPE VSTYLE

A value of this enumerated data type is used to determine the type of workpiece or tool that is displayed, e.g. by an instance of the MECH_VIEW function block type, as illustrated in the following table. The displays shown in this table were generated by using an instance of the MECH_VIEW type with POS:=75(approximately), WKPC:=COLOR#yellow, BKGD:=COLOR#blue, LEN:=5, DIA:=25, DIR:=1 and STYLE:=VSTYLE#x, where x is as shown in the table.
