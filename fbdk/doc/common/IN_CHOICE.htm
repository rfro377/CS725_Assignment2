FUNCTION_BLOCK IN_CHOICE

An instance of this type encapsulates a read-only instance of the javax.swing.JComboBox class. The strings in the drop-down list are taken from the comma-separated list at the CHOICES input. When an element is selected from the drop-down list, an IND event occurs with the string value of the selected element at the OUT output and the zero-based array index of the selected element at the I output.
