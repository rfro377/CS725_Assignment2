FUNCTION_BLOCK IN_EVENT

An instance of this service interface function block type provides a GUI pushbutton for the user to enter an event occurrence into an IEC 61499 function block network through the IND output of the function block. The LABEL1 inputs provides a label for the pushbutton. Propagation of the event through the IND output is inhibited when the QI input is false.
