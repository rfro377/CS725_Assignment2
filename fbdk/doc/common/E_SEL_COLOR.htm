FUNCTION_BLOCK E_SEL_COLOR

An instance of this type provides event-driven selection of data among COLOR inputs. Data at the OUT variable is copied from the data (IN0 or IN1, respectively) associated with the most recent event at the EI0 or EI1 input, followed by an event at the EO output.
