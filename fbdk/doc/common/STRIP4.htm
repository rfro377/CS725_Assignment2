FUNCTION_BLOCK STRIP4

An instance of this function block type encapsulates a four-channel strip chart recorder which displays the values of the inputs PEN1 - PEN4 as percent full-scale values in the range {0..100}.
  • The colors of the pens are programmable with the COLORS parameter.
  • The x-increment per occurrence of an REQ event is programmable with the DX parameter.
