FUNCTION_BLOCK MECH_VIEW

An instance of this function block type encapsulates the View element for displaying the position of a workpiece or tool along a single axis of motion.

  • The type of workpiece or tool displayed is determined by the value of the STYLE input. See the documentation of the VSTYLE data type for descriptions of the possible display styles.
  • The default value of the DIA input may be overridden when an instance of this function block type is used in an instance of the VIEW_PANEL resource type.
