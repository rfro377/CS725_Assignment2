FUNCTION_BLOCK OUT_COLOR

An instance of this function block type encapsulates a graphic component that displays a circle filled with a specified color, with a corresponding label to the right of the colored circle.
  • The color is updated upon the occurrence of an event at the REQ input.
  • See OUT_BOOL for an example of the use of an instance of this function block type.
