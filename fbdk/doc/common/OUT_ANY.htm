FUNCTION_BLOCK OUT_ANY

An instance of this function block type encapsulates a text field which can be used to display a double-byte character string contained in a variable of any elementary or derived IEC 61131-3 type, i.e., any subtype of the generic type ANY.

  • The data type name of the variable to be displayed is specified by the TYPE input.
  • An initial value to be displayed is spcified by the IVAL input.
  • The displayed string is updated to the value given by the IN input upon an occurrence of an event at the REQ input.
  • The maximum number of characters displayable without overflowing the text field is specified by the W (width) input.
