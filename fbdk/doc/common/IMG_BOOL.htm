FUNCTION_BLOCK IMG_BOOL

An instance of this function block type provides a human/machine interface (HMI) element for the entry of an IEC 61131-3 BOOL element in a similar fashion as the IN_BOOL function block type, with the additional feature that the user can use the IMG0 and IMG1 inputs to specify the icons to be displayed when the Boolean output stands at FALSE or TRUE, respectively.

The location of the image to be used is relative to the path fb/rt/hmi/ in the fbrt.jar file or the lib/fb/rt/hmi directory of the FBDK.

See the PNP_MVCL configuration for an example of the use of an instance of this type.
