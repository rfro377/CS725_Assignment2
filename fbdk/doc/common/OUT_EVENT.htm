FUNCTION_BLOCK OUT_EVENT

An instance of this function block type utilizes an instance of the OUT_BOOL type to present a colored circle with an associated label.
  • When an event occurs at the REQ input, the circle changes to the color specified by the FLASH input for a period determined by the DT input, and then reverts to the default FALSE-state color (white) for the OUT_BOOL type.
  • At the end of the period an event occurs at the CNF output.
  • The behavior of this component upon the occurrence of a sequence of events at the REQ input is determined by the behavior of the E_DELAY function block instance.
