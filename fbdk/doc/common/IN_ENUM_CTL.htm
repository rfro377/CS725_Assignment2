FUNCTION_BLOCK IN_ENUM_CTL

  • An instance of this type is used in conjunction with an instance of the IN_CHOICE type to form the composite IN_ENUM type.
  • The operation of an instance of this type is as shown in the service sequences.
  • If the value of I is outside the range of choices in a select operation, the first or last choice is selected as appropriate.
