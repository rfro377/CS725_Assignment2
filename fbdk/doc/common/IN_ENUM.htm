FUNCTION_BLOCK IN_ENUM

An instance of this type provides a drop-down list consisting of the elements of an enumerated data type as defined in IEC 61131-3.
  • The data type name of the enumerated data type is specified by the TYPE input.
  • The initial value displayed is the specified initial value for the data type.
  • When an element is selected from the drop-down list, an IND event occurs with the enumerated value of the OUT output corresponding to the selected element from the drop-down list.
