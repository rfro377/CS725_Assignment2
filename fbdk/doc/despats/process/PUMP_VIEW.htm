FUNCTION_BLOCK PUMP_VIEW

An instance of this function block type displays a View of a fluid pump showing the index position of a rotating shaft. It is implemented as a subclass of the SHAFT_VIEW type. See the TANK_VIEWL configuration for an example of its use.

The figures below show typical displays for parameter values DIA:=75 pixels, FGD:=COLOR#blue, BKGD:=COLOR#white, IX:=COLOR#red, ARC:=22 degrees, and DIR:=FALSE (outlet direction = right).
