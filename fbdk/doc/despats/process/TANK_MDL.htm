FUNCTION_BLOCK TANK_MDL

An instance of this function block type provides a Model for the filling and draining of liquid in a cylindrical tank. The semantics of the START, STOP, and DT inputs and the EO output are the same as for the E_CYCLE function block type defined in IEC 61499-1. The semantics of the other inputs and outputs are documented below. See TANK_MVL for an example of the use of an instance of this function block type.
