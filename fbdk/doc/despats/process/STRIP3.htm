FUNCTION_BLOCK STRIP3

An instance of this function block type encapsulates a three-channel strip chart recorder which displays the values of the inputs PEN1, PEN2 and PEN3 as percent full-scale values in the range {0..100}. The colors of the pens are programmable with the COLORS parameter, and the x-increment per occurrence of an REQ event is programmable with the DX parameter.

See the TANK_MVL system configuration for an example of the use of an instance of this type.
