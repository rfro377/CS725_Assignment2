7. Perform Distribution Design

This system configuration illustrates a distribution design of the TANK_MVCL example in which the Tagged Data pattern is used to establish UDP communications with a remote Model and View contained in an instance of the FRAME_DEVICE type.

TheModel+View functions can be launched locally or remotely and configured from the local FBDK by setting appropriate IP host and port addresses on the MGR_ID input of the MODEL device.
