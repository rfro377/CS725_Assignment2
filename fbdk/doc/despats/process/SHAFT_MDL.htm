FUNCTION_BLOCK SHAFT_MDL

An instance of this function block type provides a simple model for a rotating shaft. The semantics of the START, STOP, and DT inputs and the EO output are the same as for the E_CYCLE function block type defined in IEC 61499-1. The V input specifies the rotation speed in degrees per second. At each simulation interval, the ANGLE (shaft position) output is incremented by an appropriate amount as determined by the DT and V inputs.

See the TANK_MVL system configuration for an example of the use of an instance of this function block type.
