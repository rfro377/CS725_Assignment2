5. Develop and Test Controllers

As illustrated below, the TANK_MVCL system configuration utilizes instances of the FACEPLATE, STRIP4, and FB_PIDR function block types to simulate and display the Proportional/ Integral/ Derivative (PID) control of liquid level in the process control example.

Set point (SP), process variable (PV), controller output (OUT), and pump speed (PUMP) values are exchanged among resources using the local multicast pattern and are plotted on the strip chart in colors of orange, blue, green and red, respectively.

The overall structure of the system is shown as the TANK_CTL application.
