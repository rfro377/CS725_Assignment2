FUNCTION_BLOCK VIEW_HMI

An instance of this function block type provides a set of labelled HMI elements to interface with and perform static testing on any view element with a "request" event input associated with a UINT position input in the range {0..100} and a COLOR workpiece input. See BORING_HMI for examples of its usage, and BORING_TEST for examples of the associated graphical displays.
