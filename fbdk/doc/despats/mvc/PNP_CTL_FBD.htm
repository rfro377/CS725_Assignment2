FUNCTION_BLOCK PNP_CTL_FBD

An instance of this function block type implements the logic for the AUTO (automatic) mode control of a two-dimensional pick and place unit.

This block is implemented as a simple FB type with an algorithm in the IEC 61131-3 FBD language. For an alternative implementation in the LD language, see the PNP_CTL_LD function block type.
