FUNCTION_BLOCK XBAR_CTL

An instance of this function block type provides the simple combinational logic for control of a transfer bar modeled by an instance of the XBAR_MDLL type. The bar is advanced when a part is present and the bar is not yet at its END position, and is retracted back to the HOME position when a part is not present.
