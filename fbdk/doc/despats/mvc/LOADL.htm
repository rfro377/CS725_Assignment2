FUNCTION_BLOCK LOADL

An instance of this function block type provides the local multicast functionality for providing the data to simulate workpiece loading with a remote instance of the E_LDU type. Multiple instances of this type may be used with a single remote E_LDU instance; see, for instance, the DRILL_MVL  and  ASM_MVL system configurations.

The default value for the ID input (when no parameter value is explicitly connected) is the function block instance name.
