FUNCTION_BLOCK XY_MDLL

An instance of this function block type provides a Model for the operation of a two-dimensional material handling mechanism, utilizing instances of the MECH2_MDL type to model the motion of the x- and y-axes, and an instance of the E_LDU type to model the loading and unloading of a workpiece to and from the mechanism.

An instance of this type also provides the service interfaces necessary for integration of this model with other elements of the MVCD framework, using the local multicast pattern. Semantics of the *_ID inputs are defined in the documentation for the XBAR_MDLL type.

A View of this model is provided by an instance of the XY_VIEWL type.

See the PNP_MVL configuration for an example of the use of an instance of this type.
