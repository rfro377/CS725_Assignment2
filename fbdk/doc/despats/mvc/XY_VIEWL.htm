FUNCTION_BLOCK XY_VIEWL

An instance of this function block type encapsulates an instance of the XY_VIEW type and the local multicast service interfaces necessary for inter-layer communication with the View layer of the MVC framework.

See the PNP_VIEWL configuration for an example of the use of an instance of this type to represent a pick and place mechanism.

See the documentation of the MECHC_VIEWL type for usage of the x_ID inputs.
