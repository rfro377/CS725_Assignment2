FUNCTION_BLOCK CNV_MDLL

An instance of this function block type provides the service interfaces necessary for operation of  an instance of the CNV_MDL type with the local multicast pattern. Semantics of the *_ID inputs are defined in the documentation for the XBAR_MDLL type. A View of this model is typically provided by an instance of the MECH_VIEWL type.

See the TESTBED_MVL configuration for examples of the use of instances of this type.
