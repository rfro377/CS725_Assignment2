FUNCTION_BLOCK MECH1_DIAG

An instance of this function block type provides diagnostics for unidirectional motion, e.g., for one direction of a bidirectional mechanism, for example as modeled by an instance of the MECH2_MDL type. A timeout fault FLT is triggered if the mechanism fails to reach the END sensor within N occurrences of an event at the CLK input after the FWD command is received.
