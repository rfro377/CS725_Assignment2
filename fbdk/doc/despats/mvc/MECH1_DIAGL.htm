FUNCTION_BLOCK MECH1_DIAGL

An instance of this function block type provides local multicast functionality to an encapsulated instance of the MECH1_DIAG type. Diagnostic timing clock events are provided on the local channel identified by the CLK_ID input, and local publication of the diagnostic information is provided on the channel identified by the DIAG_ID input.
