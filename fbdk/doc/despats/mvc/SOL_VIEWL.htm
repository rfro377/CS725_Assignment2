FUNCTION_BLOCK SOL_VIEWL

An instance of this function block type. encapsulates an instance of the MECH_VIEW function block type, with its parameters specialized to render a representation of a solenoid mechanism. Local multicast interfaces are provided to receive position information and feed back user-injected fault conditions to a Model element such as an instance of the SOL_MDLL type. See the XBAR_MDLL function block type for documentation of the REND_ID and UI_ID inputs.
