8. Perform Mechatronic Design

This system configuration illustrates a mechatronic design of the pick and place example. This is done by modifying the distributed PNP_MVCDR configuration as follows:

  • Model+View functions for the conveyor and pick-and-place unit are partitioned into separate, distributed CNV and PNP devices, respectively.
  • Control of the conveyor is already localized in the CNV device.
  • Control of the pick-and-place unit is moved from the CONTROL.AUTO_CTL resource of the PNP_MVCDR configuration to the PNP.AUTO_CTL resource of the PNP_MVCDM configuration.
  • Diagnostics for the pick-and-place unit are moved from the DIAGNOSTICS.MONITOR resource of the PNP_MVCDR configuration to the PNP.DIAGNOSTICS resource of the PNP_MVCDM configuration.
  • HMI and mode control functions from the PNP_MVCDR configuration are allocated to the HMI device of the PNP_MVCDM configuration.
  • Tagged data is easily re-allocated in the new functional distribution, using a TAGS resource in each device.
