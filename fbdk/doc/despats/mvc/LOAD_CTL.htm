FUNCTION_BLOCK LOAD_CTL

An instance of this function block type controls the motion of a "loader" mechanism for transferring a workpiece from a "source" mechanism to an adjacent "destination" mechanism. It is assumed that the loader (for example an instance of the SOL_MDLL type) moves in the source-to-destination direction when the FWD signal is TRUE, and in the reverse direction when it is FALSE. It is also assumed that the loader itself contains mechanical stops at both extremes of the motion.
