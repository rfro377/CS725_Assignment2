FUNCTION_BLOCK MACH_CTL

An instance of this function block type provides the logic for control of spindle rotation and axial motion of a machine tool modeled, for instance, as in the DRILL_MDL resource of the MODEL device in the DRILL_MVL configuration. The state of the workpiece is tracked by the internal variable MACHINED used in the Ladder Diagram (LD) control algorithm.
