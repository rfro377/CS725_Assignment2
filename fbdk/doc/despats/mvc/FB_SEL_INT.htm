FUNCTION_BLOCK FB_SEL_INT

Upon the occurrence of an event at the REQ input of an instance of this function block, the value of the OUT variable will take on the value of the IN0 or IN1 input, depending on whether the value of the G input is FALSE or TRUE respectively, and an event will then occur at the CNF output.
