FUNCTION_BLOCK LOADLE

An instance of this function block type extends the local multicast functionality of the LOADL function block type by adding a QI input. Local publication of the workpiece loading data and event upon an occurrence of a REQ event is enabled when the QI input is true and disabled when it is false.

The default value for the ID input (when no parameter value is explicitly connected) is the function block instance name.

See the LIFTER_MVL system configuration for examples of the use of instances of this function block type.
