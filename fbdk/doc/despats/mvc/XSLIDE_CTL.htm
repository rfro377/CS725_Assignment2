FUNCTION_BLOCK XSLIDE_CTL

An instance of this function block type provides the sequential logic for control of a transfer slide with workpiece clamp modeled by an instance of the XSLIDE_MDLL type. The normal sequence of operation is implemented by the controller's Execution Control Chart (ECC).
