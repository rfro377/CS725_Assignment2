FUNCTION_BLOCK WKPC_EMIT

An instance of this function block type uses an instance of the WKPC_GEN type to produce periodically as an output a value of the VSTYLE enumerated data type, selected at random from the WS0 and WS1 inputs, at a time interval specified by the DT input.
  • See the TESTBED_MVL and TESTBED_MVCL system configurations for examples of the use of instances of this function block type.
