FUNCTION_BLOCK DSCYL_MDLL

 This function block type encapsulates an instance of the DSCYL_MDL type along with the function blocks necessary to link it to appropriate Controller and View elements via the local multicast pattern. An instance of the E_LDU type provides for the modeling of loading and unloading of a workpiece on a carrier associated with the cylinder. See the XBAR_MDLL function block type for the semantics of the *_ID inputs.
