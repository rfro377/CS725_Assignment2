FUNCTION_BLOCK XSLIDE_MDLL

An instance of this function block type serves as a model for a transfer slide with clamp by encapsulating an instance of the general-purpose bidirectional clamped motion model type MECHC2_MDLL. Loading and unloading of a workpiece at the HOME position, and modification of the workpiece at the END position, is modeled by an instance of the LRU_MDL type.

  • See the DRILL_MVL configuration for examples of the use of instances of this type.
  • Inter-layer communications with the View, Control and HMI layers of the MVC framework are provided according to the local multicast pattern.
  • See the XBAR_MDLL function block type for descriptions of the semantics of the x_ID inputs.
