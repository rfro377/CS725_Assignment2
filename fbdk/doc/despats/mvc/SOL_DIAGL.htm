FUNCTION_BLOCK SOL_DIAGL

An instance of this function block type uses two instance of the MECH1_DIAGL type to perform diagnostic functions for a solenoid device modeled by an instance of the SOL_MDLL type. In addition, sending and receiving of data using the local mulitcast design pattern are provided. Sensor and actuator data are received on the local channels identified by the SENSE_ID and ACT_ID inputs, respectively, and the diagnostic output is published on the local channel identified by the DIAG_ID input. Diagnostic clock timing events are received on the local channel identified by the CLK_ID input. Identification of the source of the diagnostic message is provided by the SRC input.
