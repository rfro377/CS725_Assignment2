FUNCTION_BLOCK DSCYL_MDL

An instance of this function block type models the operation of a dual-stroke cylinder. Two instances of the MECH2_MDL type model the operation of the two cylinders, and the ETA parameter determines the contribution of each cylinder to the total linear displacement as calculated by the WTAV_UINT block. For instance, with a value of 0.4 for ETA, the first cylinder would contribute 40% of the total stroke and the second cylinder would contribute 60%.

See the DSCYL_MDLL function block type for an example of the use of an instance of this function block type.
