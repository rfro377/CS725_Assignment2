FUNCTION_BLOCK MODE_AMS

An instance of this function block type implements the logic for three-mode AUTO/MANUAL/STOP control as an IEC 61499 Execution Control Chart (ECC).

See the PNP_MVCL configuration for an example of the use of an instance of this type.
