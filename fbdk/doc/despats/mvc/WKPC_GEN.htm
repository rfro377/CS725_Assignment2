FUNCTION_BLOCK WKPC_GEN

An instance of this function block type produces as an output a value of the VSTYLE enumerated data type, selected at random from the WS0 and WS1 inputs,  upon each occurrence of an event at the REQ input.

  • The probability of the output being selected from the WS1 input is given by the P1 input, e.g., a value of 100 at the P1 input will result in the OUT value being selected from the WS1 input 100 percent of the time.
  • See the TESTBED_MVL and TESTBED_MVCL system configurations for examples of the use of instances of this function block type.
