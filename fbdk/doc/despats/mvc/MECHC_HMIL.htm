FUNCTION_BLOCK MECHC_HMIL

An instance of this function block type includes the HMI elements and local multicast functionality to test an instance of the MECHC_VIEWL type, including both the workpiece and clamp positions. It can also be used for any view element that requires two UINT position inputs, such as the XY_VIEWL type. See the DRILL_VIEWL and PNP_VIEWL configurations for examples of its use.

The local multicast channel name, as well as the HMI label, is specified by the REND_ID input. For convenience, the default value of this input is "<name>_REND", where <name> is the instance name of the MECH_HMIL instance. This can be overridden by connection to a non-default variable or configuration parameter.
