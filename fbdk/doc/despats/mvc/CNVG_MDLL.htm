FUNCTION_BLOCK CNVG_MDLL

An instance of this function block type provides the service interfaces necessary for operation of  an instance of the CNV_MDL type with the local multicast pattern. An instance of the SOL_MDL type provides a model for the operation of a gate or stopper at the end (100%) position of the conveyor. Semantics of the *_ID inputs are defined in the documentation for the XBAR_MDLL type. A View of this model is typically provided by an instance of the MECHC_VIEWL type with the value of its CNV input set TRUE (1).

See the TESTBED_MVL configuration for examples of the use of an instance of this type.
