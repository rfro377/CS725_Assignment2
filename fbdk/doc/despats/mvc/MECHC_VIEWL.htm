FUNCTION_BLOCK MECHC_VIEWL

An instance of this function block type encapsulates an instance of the MECHC_VIEW type and the local multicast service interfaces necessary for inter-layer communication with the View layer of the MVC framework.

See the DRILL_VIEWL and LIFTER_VIEWL configurations for examples of the use of instances of this type to represent a transfer slide with clamped workpiece and a conveyor with a stopper/gate, respectively.

For convenience, the names of the local multicast channels used by an instance of this type with an instance name denoted by <name> will have the default values shown in the following table. These values can be overridden by connection to a non-default variable or configuration parameter.

Default value
