FUNCTION_BLOCK SOL_MDLL

An instance of this function block type encapsulates an instance of the SOL_MDL type to model the behavior of a solenoid mechanism with mechanical stops at each end of its motion. The local multicast pattern provides local communication of the simulation clock events, sensor inputs, actuator outputs, rendering and fault simulation data.

See the DRILL_MVL system configuration for an example of the use and testing of instances of this function block type.
See the XBAR_MDLL function block type for descriptions of the semantics of the x_ID inputs.
